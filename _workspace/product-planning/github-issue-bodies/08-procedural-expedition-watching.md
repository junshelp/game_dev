## What to build

Add Procedural Expedition Layout and Expedition Watching to the compact overlay. The prototype should generate rooms, corridors, props, and a walking path from a seed, then move a small character automatically through the scene. Dungeon activity should read as atmosphere and progress context, not as controllable combat or loot gameplay.

## Acceptance criteria

- [ ] The overlay generates a procedural arrangement of rooms, corridors, props, and a walking path.
- [ ] The same seed produces the same layout and walking path.
- [ ] The generated layout includes at least one usable walking path.
- [ ] A character automatically moves along the path without player input.
- [ ] Ambient monsters or events are visual flavor only and do not drive Gothic Pack Earning.
- [ ] No direct movement, attack, route, or skill controls are introduced.
- [ ] Layout generation behavior is covered by thin deterministic tests.
