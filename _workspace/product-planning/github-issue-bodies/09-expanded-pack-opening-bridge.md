## What to build

Connect ready packs from the Borderless Expedition Overlay into an Expanded Pack Opening View. When a Gothic Pack is ready, the Pack Navigation Button should expand the overlay upward into a medium Pack Opening Sequence view, preserve the bottom UI alignment, run the existing pack-opening flow, and return to the compact overlay after completion.

## Acceptance criteria

- [ ] Pack Navigation Button is inactive or clearly not ready when no pack is available.
- [ ] Clicking the Pack Navigation Button with a ready pack enters an Expanded Pack Opening View.
- [ ] The expanded view grows from the taskbar-adjacent overlay instead of opening a separate normal window.
- [ ] The expanded view is medium-sized and avoids fullscreen takeover.
- [ ] Existing Pack Opening Sequence behavior remains focused on Gothic Pack, Tear Gesture, Card Stack, Reveal Swipe, Rare Signal, Card Reveals, and Pack Summary.
- [ ] Completing the Pack Opening Sequence returns to the compact Expedition View.
- [ ] The ready pack count updates after a pack is opened.
